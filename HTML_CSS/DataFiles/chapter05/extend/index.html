Working with Positions

Sticky

This is an example of an element with a sticky position. Its position is based on the user's scroll position.

You must use the position property and assign it a value of sticky, to make it compatible with more browsers, you may need to add a prefix, such as -webkit-, after that, you should set the top property to 0

Relative

This is an example of an element with a relative position. It can move around a little, depending on the viewport, but it is in a position relative to itself.

You must use the position property with a value of relative, I used 0 for the top property and 0 for the left property, which left the box in the same place as it should be.

Fixed

This is an example of an element with a fixed position. Its position is relative to the viewport.

I used 90 pixels for the top property and 10 pixels for the right property

Absolute

This is an example of an element with an absolute position. It is similar to the fixed position; however, it is positioned relative to the closest parent element.

I used 0 for both the right and bottom properties and it moved the box to the bottom right of the page
